Note on the following:
Paintings:
Comments and ratings are dynamically updated. AJAX implemented.

Galleries Near you :
a. Click to search art galleries nearby - Loads location map based on geolocation and shows the art galleries nearby.
b. Search at a specific location - Type a location like San jose, sunnyvale etc and then click on "Click to search art galleries nearby". Loads the specific location map and shows the art galleries nearby.
c. Keyword - Type in a partial name of art gallery like "Gallery". It shows all the galleries whose name contains gallery.

Chat:
To run chat, table needs to be created in sql database. 
Table is given in inc folder of chat(inc folder of source inside the project folder)
